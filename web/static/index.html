Ludoforge — 게임 수치·경제 시스템 검증기

산문 기획과 스프레드시트를 넣으면 AI가 검증 모델(.lf)로 번역하고, 판정은 항상 solver가 한다 — 모순은 Z3가 증명하고(check/bmc), 분포는 Monte Carlo가 추정한다(sim).

① 기획 입력

산문으로 수치·경제 룰을 쓰고, 수치 표는 CSV로 붙인다(표는 AI를 거치지 않고 그대로 변환된다).

② 모델 확인 — 사람 게이트

왼쪽은 생성된 검증 모델(.lf, 수정 가능), 오른쪽은 같은 모델의 규칙서(산문)다. 규칙서를 읽고 기획 의도와 맞는지 확인한 뒤 실행으로 넘어간다.

③ 실행 — 증명(check·bmc) / 추정(sim)

check: 정적 모순 증명(unsat core) · bmc: 동역학 도달성/불변식/데드락 증명 · sim: 분포/확률 추정(증명 아님 · 신뢰구간 동봉).
